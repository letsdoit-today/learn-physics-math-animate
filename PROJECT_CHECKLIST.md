## 自动化检查命令

### 运行项目检查
```bash
npm run check
# 或
npm run lint
```

### 构建前自动检查
```bash
npm run build
# 会自动运行 prebuild 脚本 (npm run check)
```

### 开发模式检查
```bash
npm run dev
# 启动开发服务器前会构建和检查
```

## 问题排查指南

### 常见问题
1. **构建失败**
   - 检查 `npm run check` 输出
   - 验证演示配置文件格式
   - 检查模板语法错误

2. **页面显示异常**
   - 检查浏览器控制台错误
   - 验证CSS和JS文件路径
   - 检查模板变量传递

3. **导航功能异常**
   - 检查 `navigation.json` 格式
   - 验证URL路径配置
   - 检查面包屑生成逻辑

### 调试工具
- 浏览器开发者工具
- Node.js 调试模式
- 构建脚本日志输出
