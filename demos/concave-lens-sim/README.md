# 凹透镜成像规律演示

这是一个基于Web的凹透镜成像规律交互式演示，使用Snap.svg库实现光线追踪和动画效果。

## 功能特点

- 实时展示凹透镜成像过程
- 可调节物距，观察成像变化
- 动画演示物体移动时的成像变化
- 显示物距、像距、放大倍数等物理参数
- 展示光线路径，帮助理解成像原理

## 文件结构

- `demo.ejs` - 演示页面模板
- `demo.css` - 演示样式
- `demo.js` - 演示逻辑
- `demo.json` - 演示配置

## 使用说明

1. 调节物距滑块，观察成像变化
2. 点击"播放"按钮，观看自动动画演示
3. 点击"暂停"按钮，暂停动画
4. 点击"重置"按钮，恢复初始状态

## 物理原理

凹透镜对光线有发散作用，无论物体放在凹透镜前的什么位置，凹透镜都成正立、缩小的虚像。虚像位于透镜同侧，且像距小于焦距。
