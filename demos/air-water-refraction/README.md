# 空气-水折射与反射演示

这是一个基于Web的空气到水的折射与反射交互式演示，使用Snap.svg库实现光线追踪和动画效果。

## 功能特点

- 实时展示光线在空气和水界面的反射与折射
- 可调节入射角，观察反射角和折射角的变化
- 动画演示入射角从60°减小到0°的过程
- 显示入射角、反射角、折射角、反射比和透射比等物理参数
- 根据菲涅尔方程计算反射比和透射比

## 文件结构

- `demo.ejs` - 演示页面模板
- `demo.css` - 演示样式
- `demo.js` - 演示逻辑
- `demo.json` - 演示配置

## 使用说明

1. 调节入射角滑块，观察反射和折射现象
2. 点击"开始演示"按钮，观看自动动画演示
3. 点击"暂停"按钮，暂停动画
4. 点击"重置"按钮，恢复初始状态

## 物理原理

当光从一种介质进入另一种介质时，会发生反射和折射现象。反射角等于入射角，而折射角与入射角之间的关系遵循斯涅尔定律：n₁sin(θᵢ) = n₂sin(θₜ)，其中n₁和n₂是两种介质的折射率，θᵢ是入射角，θₜ是折射角。空气的折射率约为1.00，水的折射率约为1.33。
